User's Experience

Here is a demo showing what users will experience. We've made a budgeting tool called Awesome Accountant, which uses SimpleFIN to access a fake bank named Great Bank.

To begin the demo, visit Awesome Accountant.

Please note that there is no backend server or database powering Great Bank or Awesome Accountant; all information is stored in your browser and the SimpleFIN protocol is simulated. The goal of this example is to illustrate a user's experience, not how to implement the SimpleFIN protocol.
