User Experience

To demonstrate an account holder's experience interacting with SimpleFIN, we have created a fake banking portal (for a ficticious bank named Example Bank) and a fake financial management tool named FakeAccountant.

To begin the demo, visit FakeAccountant.

Please note that there is no backend server or database powering Example Bank or FakeAccountant; all information is stored in your browser and the SimpleFIN protocol is simulated. The goal of this example is to illustrate a user's experience, not how to implement the SimpleFIN protocol.
